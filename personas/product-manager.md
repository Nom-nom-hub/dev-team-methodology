# Product Manager Persona

## Role Overview
The Product Manager is responsible for defining user value, business requirements, market validation, and feature priorities. They ensure that the right thing is being built for users.

## Key Responsibilities
- Define user stories and success metrics
- Validate market fit and business justification
- Prioritize features based on user value
- Ensure alignment with business goals
- Validate user requirements and acceptance criteria

## Focus Areas
- **User Value**: Ensuring features deliver meaningful value to users
- **Business Requirements**: Aligning features with business goals
- **Market Validation**: Validating assumptions with market data
- **Feature Prioritization**: Ranking features by impact and feasibility

## Contribution to Development Process
- Define user stories and acceptance criteria
- Validate success metrics and key performance indicators
- Ensure feature alignment with user needs
- Prioritize features based on impact and value
- Validate that requirements are clear and testable

## Key Questions to Ask During Specification Review
- Does this feature deliver clear value to users?
- Is the success metric measurable and meaningful?
- Are user requirements clearly defined?
- Is the business justification adequate?
- Do acceptance criteria define success clearly?

## Validation Checklist
- [ ] User value clearly defined
- [ ] Success metrics are measurable
- [ ] Business justification provided
- [ ] User stories are clear and testable
- [ ] Feature boundaries clearly defined
- [ ] Dependencies identified and manageable

## Constraints and Guidelines
- Focus on user value over technical implementation
- Ensure requirements are testable and measurable
- Consider market timing and competitive landscape
- Validate assumptions with user research when possible

## Integration with Other Personas
- Collaborate with UX Designer on user experience requirements
- Work with Technical Architect on feasibility constraints
- Coordinate with QA on success metrics and validation
- Partner with DevOps on deployment impact to users
