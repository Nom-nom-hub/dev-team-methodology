# /team.personas Command

Define and document team personas for the current project.

## Process

1. Identify required team personas for the project
1. Document each persona's responsibilities and focus areas
1. Define how each persona contributes to the development process
1. Create validation checklists for each persona

## Output

- Creates persona documentation in `personas/` directory
- Generates team collaboration guidelines
- Provides cross-reference between personas
- Creates validation checklists for each persona
